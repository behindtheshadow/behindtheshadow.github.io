关于本网站你可能会感兴趣的一些信息：

  • 本网站由 Jekyll 生成。
  • 使用 Jekyll 需要一定的 HTML, CSS 和 Markdowm 语法基础。
  • 本网站的主题在 Hyde 的基础上有一些改变，特别感谢原作者 [@mdo](https://twitter.com/mdo) 的无私分享。
  • 本网站托管在免费的 GitHub Pages 上。
  • 编程推荐使用 GitHub 社区的 Atom 编辑器。
  • 对于 Markdown 文档推荐使用 Typora 编辑器。
